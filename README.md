Welcome to CS 253 - Data Structures and Algorithms!
This course is designed to provide students with a comprehensive understanding of fundamental data structures and algorithms.

We will use Java to implement various data structures and algorithms.

**I am using VS Code as my IDE and I recommend you do the same.**

# Getting Started:

Clone the repository to your local machine and run `Main.java` to make sure everything is set up correctly. If you don't know what this means or you need help, please reach out to me for assistance.

# Week 1 Overview:

We reviewed the basics of Java programming and introduced Linked Lists, Queues, Stacks, Trees (very simple tree).

# Week 1 Tasks:

Re-implement the files in `lab02` to use `String` instead of `int`. This will help you get familiar with the code and understand how to work with different data types in Java.
If you prefer to use generic types, you can modify the code to use generics instead of `String`. This will allow you to work with any data type and will be a good exercise in understanding how generics work in Java.
